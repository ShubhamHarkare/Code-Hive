## Features

- Real-time collaborative coding
- Room-based workspaces
- Python syntax highlighting with CodeMirror
- Live code updates across multiple users
- Copy room ID and invite teammates easily
- Clean and simple UI

---

## Tech Stack

- **Frontend:** React, CodeMirror, React Router
- **Backend:** Node.js, Express, Socket.IO
- **Languages:** JavaScript
- **Deployment:** Can be hosted on any Node + React environment

## Installation

### Clone the repository

```bash
git clone https://github.com/ShubhamHarkare/Code-Hive.git
cd Code-Hive
```

## Server
```bash
cd server
npm install
npm start
```

## Client
```bash
cd client
npm install
npm start

```
